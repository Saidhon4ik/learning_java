# Java Audio Player

## Overview

This is a simple command-line audio player written in Java using the standard audio API.

The program allows basic interaction with an audio file through console input.

---

## How to Run

1. Make sure Java is installed
2. Add your `.wav` file and update the path in the code:

```
String filePath = "C:\\path\\to\\your\\file.wav";
```

3. Compile and run:

```
javac Main1.java
java Main1
```

---

## Notes

* Supports only `.wav`, `.au`, `.aiff` formats
* Audio file must be in PCM format
* Runs in console (no GUI)
