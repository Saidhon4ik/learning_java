# Hangman (Java)

## Overview

A simple command-line implementation of the classic Hangman game written in Java.

The player tries to guess a hidden word by entering letters, with a limited number of attempts.

---

## How to Run

1. Make sure Java is installed
2. Compile and run:

```bash
javac Main.java
java Main
```

---

## Notes

* Runs in the console (CLI)
* Input is case-insensitive
* Game ends when the word is guessed or attempts run out
