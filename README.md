## ✨ Additional Java OOP Practice

### 📌 Overview
This section contains multiple small Java programs created to practice core Object-Oriented Programming concepts and fundamental Java features 😊  
Each block represents a focused exercise rather than a full-scale application 🧠  

---

### 🧠 Concepts Covered
- Constructors & Overloading  
- Inheritance  
- Method Overriding (@Override)  
- Polymorphism  
- Abstraction  
- Interfaces  
- Encapsulation (Getters & Setters)  
- Static keyword  
- Aggregation & Composition  
- Arrays & ArrayLists  
- Exception Handling  
- File Handling (Read/Write) 📂  

---

### 🔥 Notable Examples

#### 📌 OOP Basics
- Creating classes and objects  
- Working with multiple instances  
- Method usage and object behavior  

---

#### 📌 Inheritance & Polymorphism
- Parent-child class relationships  
- Dynamic method calls using base references ⚙️  

---

#### 📌 Interfaces & Abstraction
- Implementing multiple interfaces  
- Abstract classes with overridden methods  

---

#### 📌 Aggregation vs Composition
- "has-a" relationship between objects  
- Example: Car and Engine structure 🚗  

---

#### 📌 File Handling
- Writing files using `FileWriter`  
- Reading files using `BufferedReader` 📄  

---

### 🗂️ Project Structure
All examples are written inside the `Main` class as separate practice blocks.  
Most of them are commented out and can be tested individually.
